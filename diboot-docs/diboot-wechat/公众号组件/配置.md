# 配置

## 配置参数

* 在引入了[weixin-mp]()项目模块的项目的系统配置文件中（如果开发环境与生成环境的配置参数不同，则需要分开配置），配置如下配置项的参数：

```properties
# wechat mp config
wechat.appId=
wechat.secret=
wechat.token=
wechat.aesKey=
wechat.state=

wechat.host.name=
wechat.host.context=
```

## 各配置项含义

### wechat.appId

* 公众平台的[appId]()配置。

### wechat.secret

* 公众平台的[secret]()配置。
* 该配置需要将secret通过diboot-framework中加密工具加密后存入该处。

### wechat.token

* 公众平台消息[token]()配置。

### wechat.aesKey

* 公众平台消息秘钥配置.

### wechat.state

* 回调授权中用于校验合法链接的state参数，可设置一个随机字符串

### wechat.host.name

* 该应用部署的域名名称，如[test.diboot.com]()

### wechat.host.context

* 该应用的访问前缀，比如[/rest]()
